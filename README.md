# ytwall — Видео-обои с YouTube для Windows 11

Десктоп-приложение для Windows 11: вставляете ссылку на видео из YouTube,
приложение скачивает его, складывает в библиотеку «Музыка/Клипы» — и одной
кнопкой ставит выбранный клип как **живые обои рабочего стола** (видео играет
за иконками рабочего стола, как в Wallpaper Engine / Lively).

Когда вы не на рабочем столе (открыта игра в полноэкранном режиме, видео,
презентация, компьютер заблокирован, или окно поверх закрывает рабочий стол),
обои **автоматически ставятся на паузу** — это экономит GPU и батарею.

![ytwall icon](assets/icon.png)

## Возможности

- 🎬 **Скачивание с YouTube** — на базе `yt-dlp`, выбор качества (720p / 1080p / 1440p / 4K / best).
- 🎵 **Библиотека клипов** — превью, метаданные, удаление, открытие в проводнике.
- 🖼️ **Живые видео-обои** — рендер за иконками рабочего стола через классический трюк `Progman/WorkerW`.
- ⏸️ **Умная пауза**:
  - полноэкранное приложение / игра / видео,
  - режим презентации / «Не беспокоить»,
  - блокировка / выход из системы,
  - окно поверх закрывает рабочий стол,
  - работа от батареи (опционально).
- 🔊 Регулируемая громкость (по умолчанию выключена — это ведь обои).
- 🚀 Опциональный автозапуск с Windows.
- 🌙 Современный тёмный UI на PySide6 (Qt6) с фирменным violet→pink акцентом.
- 🧊 Иконка в системном трее: пауза / возобновить / снять обои / открыть.

## Сборка `.exe`

Требования: **Windows 10/11**, **Python 3.11+**.

```powershell
git clone https://github.com/doris1372/yt-wallpaper.git
cd yt-wallpaper
powershell -ExecutionPolicy Bypass -File scripts\build_exe.ps1
```

Скрипт создаст виртуальное окружение, установит зависимости и соберёт
`dist\ytwall\ytwall.exe` через PyInstaller.

> **Важно:** для воспроизведения видео нужен **libmpv** для Windows.
> Скачайте архив с https://sourceforge.net/projects/mpv-player-windows/files/libmpv/
> и положите `mpv-2.dll` в корень репозитория **до** запуска скрипта сборки.
> Без `mpv-2.dll` UI запустится, но сами обои не заработают.

## Запуск без сборки (для разработки)

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m ytwall
```

## Как это работает

1. **Скачивание.** `yt-dlp` парсит страницу YouTube, выбирает лучший mp4/m4a
   поток в пределах нужного качества, скачивает и склеивает их через ffmpeg в
   `mp4`. Метаданные и обложка сохраняются.
2. **Видео-обои.** При активации клипа приложение посылает Progman сообщение
   `0x052C` (классический недокументированный приём), Windows создаёт окно
   `WorkerW` между фоном рабочего стола и иконками. Мы делаем `SetParent`
   нашего безрамочного Qt-окна в этот `WorkerW` — теперь оно рендерится
   позади иконок, но получает GPU-ускорение через libmpv.
3. **Оптимизация.** Каждые 500 мс приложение проверяет:
   - `SHQueryUserNotificationState` (D3D fullscreen, презентация, busy, locked),
   - класс и геометрию активного окна (полноэкранное окно поверх),
   - состояние батареи (`psutil.sensors_battery`).

   Если хотя бы одно условие срабатывает — `mpv.pause = True`. Когда вы
   возвращаетесь на рабочий стол, воспроизведение продолжается с того же кадра.

## Структура проекта

```
yt-wallpaper/
├── src/ytwall/
│   ├── __main__.py        # точка входа: python -m ytwall
│   ├── app.py             # бутстрап QApplication + Controller
│   ├── config.py          # загрузка/сохранение настроек
│   ├── library.py         # библиотека клипов (JSON-метаданные)
│   ├── downloader.py      # обёртка yt-dlp в QRunnable
│   ├── wallpaper.py       # WorkerW reparent + libmpv
│   ├── pause_monitor.py   # детектор «не на рабочем столе»
│   ├── autostart.py       # реестр HKCU\…\Run
│   ├── tray.py            # иконка в трее
│   ├── styles.py          # тёмная QSS-тема
│   └── widgets/           # вкладки UI
├── assets/                # иконки
├── scripts/build_exe.ps1  # PowerShell-скрипт сборки
├── ytwall.spec            # PyInstaller spec
├── pyproject.toml
├── requirements.txt
└── README.md
```

## Пользовательские данные

- настройки: `%APPDATA%\ytwall\settings.json`
- библиотека: `%APPDATA%\ytwall\library.json`
- лог: `%APPDATA%\ytwall\ytwall.log`
- скачанные клипы по умолчанию: `%USERPROFILE%\Videos\ytwall\`

## Зависимости

| Библиотека | Назначение |
|---|---|
| PySide6 | UI (Qt 6) |
